# MedicalKnowledge
医疗知识库 RAG：包含内科常识、用药禁忌、体检报告解读问答，具体内容请自行添加

项目说明：

models：一些常量，基本上用的都是阿里的模型，DASHSCOPE_API_KEY 参数是大模型的Key（添加到环境变量的名），您需要自己申请，

cutting.py：主程序，包含初次对整个文件夹内容的向量化，和向量化后的faiss库（轻量级）加载，运用递归方式切割文本，基本上都是基于langchain框架

new_cutting.py：是增量文件的处理，把文件放在new文件夹即可，这里注意两点，第一里面有个文件是否已经处理过验证，当前验证是使用文件的文件的哈希值方式，也可以选用存入关系数据库方式，在mysql_data.py文件里已有自定义函数，也可以直接替换。第二点，我在测试时使用了doc文件，发现一些处理的问题，后来把代码都去掉了，直接把doc文件转成docx直接用的，如果有同学有兴趣可以试着件容一下。
